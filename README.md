# Blog Frontend

This project is a blog frontend made using Vite, React 19, TypeScript, Vitest, Testing Library, TailwindCSS 4, Eslint and Prettier.

It is designed to work in conjunction with my [Blog API](https://github.com/JacobArthurs/blog-api), ensuring a full-stack application experience.

## Run

```bash
npm run dev
```

## Test

```bash
npm run test:ui
```
